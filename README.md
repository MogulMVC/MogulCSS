MogulCSS
---

MogulCSS is a Sass based CSS library.

Install
----
To install you can download from GitHub or use Bower.

`bower install mogul-css`

Demo
---

Visit MogulMVC to see a list of MogulCSS's styles.

[Demo](http://mogulmvc.com/css/)
